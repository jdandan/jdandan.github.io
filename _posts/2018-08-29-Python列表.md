> ### append

| 格式            | 描述         |
| --------------- | ------------ |
| 列表.append(值) | 在末尾追加值 |

```python
list1 = [13, 5]

# 在列表结尾追加8
list1.append(8)
print(list1)  # [13, 5, 8]
```

> ### insert

| 格式                  | 描述                               |
| --------------------- | ---------------------------------- |
| 列表.insert(索引, 值) | 在指定位置插入值, 超过索引会追加值 |

```python
list1 = [13, 5, 8]

# 在 索引1的位置 插入数据20
list1.insert(1, 20)
print(list1)  # [13, 20, 5, 8]

# 在 索引10的位置 插入数据4(没有索引10, 追加4)
list1.insert(10, 4)
print(list1)  # [13, 20, 5, 8, 4]
```
